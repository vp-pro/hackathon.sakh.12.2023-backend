### Описание Backend-части проекта:

Backend-часть проекта представляет собой веб-сервер, написанный на языке JavaScript с использованием фреймворка Express. Сервер предоставляет API для выполнения анализа текстовой информации с использованием предварительно обученных моделей и библиотек.

### Основные функции API:

1. **Определение языка текста:**
   - **Метод:** `POST`
   - **Эндпоинт:** `/v1/language_detection`
   - **Параметры:** `messages` - массив текстовых сообщений на разных языках.
   - **Возвращаемый результат:** Массив объектов с определенным языком для каждого сообщения.

2. **Функциональный анализ тональности:**
   - **Метод:** `POST`
   - **Эндпоинт:** `/v1/functional_sentiment`
   - **Параметры:** `messages` - массив текстовых сообщений для анализа.
   - **Возвращаемый результат:** Массив объектов с функциональной тональностью для каждого сообщения.

3. **Эмоциональный анализ тональности:**
   - **Метод:** `POST`
   - **Эндпоинт:** `/v1/emotional_sentiment`
   - **Параметры:** `messages` - массив текстовых сообщений для анализа.
   - **Возвращаемый результат:** Массив объектов с эмоциональной тональностью для каждого сообщения.

4. **Комбинированный анализ:**
   - **Метод:** `POST`
   - **Эндпоинт:** `/v1/sentiment`
   - **Параметры:** `messages` - массив текстовых сообщений для анализа.
   - **Возвращаемый результат:** Массив объектов с функциональной и эмоциональной тональностью, а также определенным языком для каждого сообщения.

### Зависимости:

- **Node.js**
- **Express**
- **Python 3.x**
- **Python библиотеки:**
  - `langdetect`
  - `langcodes`
  - `transformers`
  - `torch`
  - `natasha`
  - `dostoevsky`

### Инструкции по запуску:

1. **Установка Node.js и зависимостей:**
   ```bash
   npm install
   ```

2. **Запуск сервера:**
   ```bash
   node app.js
   ```

3. **Использование API:**
   - Взаимодействие с API происходит через HTTP-запросы к соответствующим эндпоинтам.

### Примечание:

Backend-часть проекта предоставляет удобный способ интеграции с предварительно обученными моделями для анализа текстовой информации, позволяя выполнять различные виды анализа на основе переданных текстовых данных.
